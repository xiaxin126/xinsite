1、各定义说明：
a.流程对象：申请的表单对象，可以是一个表或者一个主表多个副表
b.流程栏目：点击的栏目是一个申请流程，需要申请人填写表单，提交后
经一系列的审批过程，需要配置一个流程对象
c.流程任务：审批人审批环节的定制，及审批表单的定制
d.一个流程对象可以配置到多个流程栏目，一个流程栏目可以配置多个流程任务

2、一个流程对象可以配置多个流程任务，主要作用如下：
a.如果一个流程对象配置到两个栏目，那么两个栏目的流程申请
可以配置不同的流程任务
这两个栏目的申请表单是一样的，但流程任务不一样
b.一个流程对象配置到一个栏目，这个栏目有多个流程任务
申请人根据优先级，只能匹配到一个流程任务执行
c.混合使用，a与b

3、审批结点“其他配置”注意事项：
a.通过策略中设置了“审批意见字段”，在“审批表单”选项卡中，生成页面后，该字段后有一个选择框
b.修改表单设置后，该环节的审批人可以修改申请人的表单信息，修改信息是即时修改，没有保存按钮

4、其他注意事项：
a.一个流程任务只有一个“开始结点”，多个“任务结点”，一个“结束结点”
b.设置流程图，必须将任务点全部定义好，在“设置流程图”中，只能定义任务路径，及路径属性
c.流程图发布，将正式用于流程申请，申请流程后，流程任务将匹配最后一版发布的“流程图”
