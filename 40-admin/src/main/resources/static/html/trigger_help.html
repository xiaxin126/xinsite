触发器调用函数有两种方法：
1.Bean调用示例：testTrigger.testTask(#task_id, #alloc_id, "admin")
2.Class类调用示例：com.xinsite.flow.listener.task_trigger.TestTrigger("admin")
3.参数常量：字符串，布尔类型，长整型，浮点型，整型
4.参数变量：#task_id，#alloc_id，#release_id，#trigger_id，#status_id，#user_id
1)、#task_id：当前任务结点Id，wd_task表主键
2)、#alloc_id：审批任务Id，wd_task_alloc表主键
3)、#release_id：流程任务发布Id，wd_release表主键
4)、#trigger_id：任务触发器Id，wd_task_trigger表主键
5)、#status_id：流程任务状态Id，wd_task_status表主键
6)、#user_id：当前登录用户Id，sys_user表主键
5.触发器定义函数：如果返回字符串，则写入日志，也可以在定义函数中写入日志，返回空字符串，或者不返回
