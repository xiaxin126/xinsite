1、选定数据对象可以新增“主表”；选定数据列表中的一个主表单，右击可以新增“副表”。

2、一个数据对象只能有一个主表，可以有多个副表。树形列表与副表不能新增副表。

3、在js文件中用"数据库表Key"，表示数据库表名，在自己的数据库中"数据库表Key"与"数据库表名"相对应。

4、主表与副表关系可以是一对一、一对多的关系。

5、对象只能有一个附件列表，附件列表不需要任何额外定义。

6、“数据主表”对应的栏目有数据权限，必须包含“创建人”、“机构Id”、“部门Id”。

7、数据表不带“删除标识”则物理删除，数据不进回收站。

8、回收站、历史记录中所有数据表以“标题字段”显示，进回收站的数据表必须有“标题字段”。

9、所有数据表带有“主键”自增，树形表带有“父结点”，字段名称不可改。

10、主表与一对多表必须有“排序号”，字段名称不可改。
