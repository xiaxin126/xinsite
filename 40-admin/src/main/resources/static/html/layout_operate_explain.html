1、单元格编辑、行编辑：

1.点击“显示列表选择”，定义要显示的列
2.在“显示列表配置”选项卡中，双击行，对显示列表的属性配置
3.点击“界面预览”，查看设置的显示列表页面
4.点击“代码生成”，基本默认即可，如果以前生成过其他类型的表单，请先点击“清空”


2、单表窗口编辑、选项卡编辑：

1.点击“显示列表选择”，定义要显示的列
2.在“显示列表配置”选项卡中，双击行，对显示列表的属性配置
3.在“编辑窗口配置”选项卡中，对表单的主界面进行配置，点击界面配置列表
4.在“界面配置”窗口，确定生成的列数，再点击“表单items生成”，根据生成的列数，按列表顺序从左到右生成
5.如果要在界面中添加文字说明，可以点击“添加文字说明”按钮，默认添加在最后一行，可以拖动到相应位置，重新“表单items生成”
6.添加“窗口属性配置”、“选项卡属性”，基本默认配置就可以
7.添加“表单属性配置”，基本默认配置就可以
8.点击“界面预览”，不满意拖动列表顺序或者调节生成类型，再点击“表单items生成”，直到满意为止
9.在“基本配置”中，点击“界面预览”，查看设置的显示列表页面
10.点击“代码生成”，基本默认即可，如果以前生成过其他类型的表单，请先点击“清空”
