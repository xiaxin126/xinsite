1、附件列表的字段不能进行新增，字段数是固定的，只能查看。

2、除了附件列表的其他数据库表字段只能新增。

3、新增好的字段只能修改字段中文名与字段类型，字段名称不可以修改。

4、字段类型的修改有些特殊规定，比如varchar的字段长度只能从小到大的修改；varchar能变成text类型，char能变成varchar与text类型反之不可以。

5、下面列出字段类型在修改时，能进行的转换有那些：

a：char、nchar、varchar、nvarchar的字段长度：小->大
b：int->decimal
c：char->nchar、varchar、nvarchar、varchar(max)、nvarchar(max)、text、ntext
d：nchar->varchar、nvarchar、varchar(max)、nvarchar(max)、text、ntext
e：varchar->nvarchar、varchar(max)、nvarchar(max)、text、ntext
f：nvarchar->varchar(max)、nvarchar(max)、text、ntext
g：text->varchar(max)、nvarchar(max)
h：ntext->varchar(max)、nvarchar(max)
i：varchar(max)->varchar(max)、nvarchar(max)、text、ntext
j：nvarchar(max)->varchar(max)、nvarchar(max)、text、ntext

注意：
1、主表不能从char、nchar、varchar、nvarchar字段转换成varchar(max)、nvarchar(max)、text、ntext字段，主表中的text字段另存一个表；
2、修改字段时，如果该表中该字段有为空的数据，只能允许为空，请自行判断(可以人工将该字段为空的数据赋为空值，int型赋0)；

6、字段名自动生成条件: 新增字段 + 勾选“自动生成”。
