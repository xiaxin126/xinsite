MyBatis生成成功后，将生成的压缩包解压，解压包名对应数据库源，源包下有5个文件包。
1.5个文件包中，都有相同文件数的数据库表
2.将mapping包下的所有文件，拷贝到10-base>>12-mybatis>>resources>>mappings/*对应的文件夹下
3.将bll包下的所有文件，拷贝到10-base>>12-mybatis>>mybatis>>datasource/*/bll对应的文件夹下
4.将entity包下的所有文件，拷贝到10-base>>12-mybatis>>mybatis>>datasource/*/entity对应的文件夹下
5.将mapper包下的所有文件，拷贝到10-base>>12-mybatis>>mybatis>>datasource/*/mapper对应的文件夹下
6.将service包下的所有文件，拷贝到10-base>>12-mybatis>>mybatis>>datasource/*/service对应的文件夹下
