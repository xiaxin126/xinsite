1、单元格编辑：

a.输入框：复选组选择框时
列类型:checkgroupcolumn；启用选择：true
b.输入框：单选组选择框时
列类型:radiogroupcolumn；启用选择：true
c.输入框：复选框时
列类型:checkcolumn；启用选择：true。显示列表数据类型：bool


2、行编辑：

a.输入框：复选组选择框时
列类型:checkgroupcolumn；启用选择：false
b.输入框：单选组选择框时
列类型:radiogroupcolumn；启用选择：false
c.输入框：复选框时
列类型:checkcolumn；启用选择：false。显示列表数据类型：bool


3、树形列表(单元格编辑，窗口编辑)：

列表配置中，第一列请设置列类型:treecolumn；
说明：只有xtype= 'treecolumn'的列才会显示成树目录结构
