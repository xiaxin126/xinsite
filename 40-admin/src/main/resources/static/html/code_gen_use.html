代码生成是在“对象生成配置”->“配置生成”中，生成成功后，在数据对象定义中可以查看与下载对应的代码。
1.一个对象会生成7-8个文件，流程对象多一个流程列表JS
2.“创建数据表Sql”文件，打开查看窗口，拷贝sql代码，在数据库中创建对象对应的数据库表
3.“栏目点击方法JS”文件，打开查看窗口，拷贝JS代码，粘贴到“itemclick.js”文件末尾，方法名称对应菜单栏目中的“方法/地址”字段
4.“JS、流程申请JS”文件，打开下载窗口，下载文件将其拷贝到javascript>>menu>>build_form||build_flow文件夹下，build_form是普通对象，build_flow是流程对象
5.“流程列表JS”文件，打开下载窗口，下载文件将其拷贝到javascript>>menu>>build_flow文件夹下
6.“Controller”文件，打开下载窗口，下载文件将其拷贝到40-admin>>java>>controller>>build_form||build_flow文件夹下
7.“BLL”文件，打开下载窗口，下载文件将其拷贝到20-core>>java>>bll>>build_form||build_flow文件夹下
8.“数据导入文件”，*.xinsite文件格式，下载后，利用“导入表单数据”->“生成数据表”栏目数据导入功能，将设计表相关数据导入到项目数据库
9.“所有文件压缩打包Zip”，所有生成的文件打包压缩，一次性下载到本地
